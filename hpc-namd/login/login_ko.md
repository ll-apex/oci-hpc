# 로그인

## 소개

이 실습에서는 유저 이름으로 opc를 사용하여 로그인합니다.

예상 실습 시간: 2분

## 태스크: 로그인

유저 이름으로 opc를 사용하여 로그인합니다.

    <copy>
    ssh {username}\@{bm-public-ip-address} -i id_rsa
    </copy>
    

Resource Manager를 사용 중인 경우 출력에서 전용(private) 키를 가져와서 로컬 시스템에 저장합니다.
