# スタックの実行

## 概要

この演習では、スタックを実行します。

推定ラボ時間: 2分

## タスク: スタックの実行

スタックが作成されたので、ジョブを実行できます。

1.  作成したスタックを選択します。
    
2.  「Terraformアクション」ドロップダウン・メニューで、「適用」を実行してGPUインフラストラクチャを起動します。 ![](./images/tf_actions.png)
