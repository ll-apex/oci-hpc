# 스택 실행

## 소개

이 연습에서는 스택을 실행합니다.

예상 실습 시간: 2분

## 작업: 스택 실행

스택이 생성되었으므로 작업을 실행할 수 있습니다.

1.  생성한 스택을 선택합니다.
    
2.  "Terraform 작업" 드롭다운 메뉴에서 '적용'을 실행하여 GPU 인프라를 실행합니다. ![](./images/tf_actions.png)
