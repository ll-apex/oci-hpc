# 執行堆疊

## 簡介

在此實驗室中，您將執行堆疊。

預估實驗室時間：2 分鐘

## 作業：執行堆疊

現在建立堆疊了，您可以執行工作。

1.  選取您建立的堆疊。
    
2.  在「Terraform 動作」下拉式功能表中，執行「套用」以啟動 GPU 基礎架構。 ![](./images/tf_actions.png)
